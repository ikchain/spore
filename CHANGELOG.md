# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-04-01

### Added
- Core stats system: DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK (0-100)
- 5 evolution stages: SPORE → SPRITE → WRAITH → SPECTER → PHANTOM
- ASCII sprites for each evolution stage
- Contextual dialogue system with 80+ lines across all stages
- Persistent state in ~/.spore/companion.json
- JSONL event log with automatic rotation
- Weekly automatic backups
- CLI commands: status, stats, log, feed, reset, install-hooks
- Git post-commit hook integration
- Diminishing returns on stats near extremes
- MIT license, open source ready
